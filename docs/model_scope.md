# Target FP&A Forecasting Model – Scope

## Objective
Build a simplified FP&A forecasting and budgeting model using Target Corp financials to evaluate business performance and support scenario-based decision making.

## Forecast Outputs (Quarterly)
- Revenue
- Gross Margin %
- Operating Income

## Time Horizon
- Historical: Last 12–16 quarters
- Forecast: Next 4 quarters

## Key Drivers (Initial)
- Consumer demand (sales growth)
- Gross margin assumptions
- Operating expense growth

## Use Cases
- Base / Upside / Downside scenario planning
- Budget vs Actual variance analysis
- Executive-level financial insights
