## Project Overview
This project simulates an FP&A forecasting and scenario analysis workflow for Target Corporation. Using publicly available financial filings, I built a quarterly historical dataset, developed a base-case forecast, and evaluated upside and downside scenarios to assess revenue and profitability sensitivity.

## Key Skills Demonstrated
- Financial statement analysis
- FP&A forecasting and budgeting
- Scenario analysis and operating leverage
- Assumption-driven modeling
- Executive-level financial communication

## Project Structure
- Historical actuals built from Target 10-K and earnings releases
- Base-case forecast for 2025 using explicit growth and margin assumptions
- Scenario analysis to evaluate upside and downside risk
- Executive insight summary highlighting business implications
